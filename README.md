# ChatGPT PZ8

ChatGPT PZ8 is a Streamlit-based web application that allows users to interact with a custom AI assistant powered by OpenAI's API. The application is designed to provide a conversational interface for users to ask questions and receive intelligent responses.

## Features

- **Interactive Chat Interface**: Users can chat with the AI assistant in real-time.
- **Session History**: Maintains a history of the conversation for context-aware responses.
- **Customizable AI Deployment**: Integrates with Azure OpenAI Service for custom deployments.
- **Environment Configuration**: Uses `.env` files for secure and flexible configuration.

## Prerequisites

- Python 3.9 or higher
- An Azure OpenAI Service deployment
- API keys for Azure OpenAI and Lasso Security
